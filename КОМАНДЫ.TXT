Установка Python (если ещё не установлен):
cmd
# Скачать и установть Python 3.12 (автоматически добавит в PATH)
curl -o python-installer.exe https://www.python.org/ftp/python/3.12.0/python-3.12.0-amd64.exe
python-installer.exe /quiet InstallAllUsers=1 PrependPath=1
Проверка установки:
cmd
python --version
python -m pip --version

Вот пошаговая инструкция по запуску проекта Sin AI через командную строку:

1. Установка необходимых компонентов
bash
# Убедитесь, что у вас установлен Python 3.8+
python --version

# Если нет - скачайте с python.org
2. Настройка виртуального окружения
bash
# Перейдите в папку проекта
cd путь_к_папке_проекта

# Создайте виртуальное окружение
python -m venv venv

# Активируйте его:

# Для Windows:
.\venv\Scripts\activate

# Для Linux/Mac:
source venv/bin/activate
3. Установка зависимостей
bash
pip install -r requirements.txt
4. Настройка окружения
bash
# Создайте файл .env (если его нет)
# Можно скопировать пример (если есть) или создать вручную:
echo "DEEPSEEK_API_KEY=ваш_ключ" > .env
5. Запуск проекта
bash
# Основной запуск:
python main.py

# Или с дополнительными параметрами:
python main.py --debug  # режим отладки
python main.py --monitor  # с включенным мониторингом
6. Работа с интерфейсом
После запуска вы увидите приглашение:

Sin>
Доступные базовые команды:

chat [сообщение] - начать диалог

learn [файл] - обучить на файле

list - показать модели

help - справка по командам

exit - выход

7. Первые шаги (пример)
bash
Sin> chat Привет! Как тебя зовут?
Sin> learn data/documents/example.txt
Sin> status
Sin> exit
Возможные проблемы и решения:
Ошибка отсутствия модулей:

bash
pip install -r requirements.txt  # переустановите зависимости
Ошибка с API ключом:

Убедитесь, что в .env указан верный ключ DeepSeek

Если ключа нет, некоторые функции будут ограничены

Проблемы с запуском на Windows:

bash
# Если не работает venv:
python -m pip install --user virtualenv
python -m virtualenv venv
Не хватает файлов данных:

Создайте папки вручную:

bash
mkdir data data/documents data/learned data/temp
Для остановки программы используйте Ctrl+C в командной строке.
