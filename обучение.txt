 Процесс обучения (шаг за шагом)
Вы вводите команду:

bash
learn data/documents/my_file.txt
Система обрабатывает файл:

Проверяет существование файла

Определяет тип (txt/docx/pdf)

Извлекает текст (_extract_text_from_* методы)

Подготовка к обучению:

Текст разбивается на чанки (~1000 слов)

Создается временный TextDataset

Инициализируется визуализатор обучения

Обучение модели:

Происходит без перезапуска текущей модели

Веса GPT2-модели обновляются через backpropagation

Прогресс отображается в консоли

После обучения:

python
self.level_system.add_experience(20)  # Начисление опыта
self.is_learning = False  # Снятие флага обучения
2. Что происходит с моделью
Автосохранение: Нет, модель не сохраняется автоматически после обучения. Для сохранения нужно явно выполнить:

bash
save my_updated_model
Состояние модели:

Все изменения остаются в оперативной памяти

Модель продолжает работать с обновленными весами

При перезапуске main.py изменения теряются, если не было сохранения

3. Как работает сохранение
Ручное сохранение:

bash
save my_trained_model
Создается папка data/learned/models/my_trained_model

Сохраняются:

Веса модели (model_weights.pt)

Токенизатор

Метаданные (дата, параметры)

Загрузка сохраненной модели:

bash
load my_trained_model
4. Технические детали
Где хранятся изменения:

В объекте SinNetwork:

python
self.model  # Обновленные веса GPT2
self.tokenizer  # Возможны изменения при обучении на новом vocab
Безопасность:

Все операции идут в том же процессе

Нет риска повредить оригинальную модель

Можно делать несколько последовательных learn без сохранения

5. Пример полного цикла
python
# 1. Загрузка и обучение
sin.learn_from_file("data/documents/python_guide.txt")

# 2. Проверка обучения (без сохранения)
response = sin.communicate("Что такое декоратор в Python?")
print(response)  # Уже использует новые знания

# 3. Явное сохранение
sin.save_model("python_expert")

# 4. Теперь можно перезапустить программу
# и загрузить обученную версию
6. Рекомендуемый workflow
Обучение на нескольких файлах:

bash
learn doc1.txt
learn doc2.pdf
learn_code example.py python
Тестирование изменений:

bash
chat "Задайте вопрос по изученным материалам"
Когда довольны результатом:

bash
save final_model
Для постоянного использования:

bash
python main.py --model final_model
