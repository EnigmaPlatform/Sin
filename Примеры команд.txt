1. Установка Python и настройка окружения (CMD/PowerShell)

cmd

:: Проверить установленный Python
python --version
python -m pip --version

:: Если Python не установлен, скачать с python.org и установить с опцией "Add to PATH"

:: Создание виртуального окружения
python -m venv venv

:: Активация окружения
venv\Scripts\activate

:: Установка зависимостей
pip install -r requirements.txt
2. Основные команды для работы с Sin AI
Запуск приложения:

cmd
python main.py

______________________


Команды с аргументами:

cmd
:: Запуск с конкретной моделью
python main.py --model my_model_v1

:: Включение режима отладки
python main.py --debug

:: Запуск сервера мониторинга
python main.py --monitor --monitor-port 8001

:: Отключение плагинов
python main.py --no-plugins

________________________________

3. Команды внутри CLI интерфейса Sin AI
Основные команды:

chat [сообщение]          - Общение с Sin AI
learn [путь_к_файлу]      - Обучение из файла
learn_code [язык] [код]   - Обучение на примере кода
query [запрос]            - Запрос к DeepSeek API
save [имя_модели]         - Сохранение текущей модели
load [имя_модели]         - Загрузка модели
delete [имя_модели]       - Удаление модели
list                      - Список сохраненных моделей
status                    - Показать статус обучения
plugins                   - Список загруженных плагинов
exit                      - Выход из программы
Команды для работы с личностью:

personality [режим]       - Смена личности (scientist|friend|neutral)
Команды разработки:

propose                   - Предложить новый функционал
train [тип]               - Запуск тренировки (programming|communication)
